# カードショップ EC サイト ご提供にあたって

お客様専用のカードショップサイトを開設するにあたり、以下の情報をご準備ください。
ご記入いただいた内容をもとに、サイトを構築・納品いたします。

---

## ご記入シート

### 1. ショップ基本情報

| # | 項目 | ご記入欄 | 補足 |
|---|------|---------|------|
| 1 | **ショップ名** | | サイトのタイトルやロゴ横に表示されます |
| 2 | **ショップの説明文**（1〜2文） | | Google検索の結果に表示されます |
| 3 | **使用するドメイン** | | 例: `mycardshop.com`（お持ちでなければご相談ください） |
| 4 | **お問い合わせ用メールアドレス** | | サイトのフッターや注文メールに表示されます |
| 5 | **電話番号**（任意） | | 表示しない場合は空欄 |
| 6 | **住所**（任意） | | フッターに表示。例: `Tokyo, Japan` |

### 2. ロゴ・画像素材

以下の画像をご用意ください。形式は JPG または PNG で大丈夫です。

| # | 素材 | 推奨サイズ | 補足 |
|---|------|----------|------|
| 1 | **ショップロゴ** | 横200 × 縦60px 程度 | サイト左上に表示されます |
| 2 | **トップページのメイン画像**（1〜3枚） | 横1920 × 縦800px | ショップの雰囲気が伝わる画像。商品写真やバナーなど |
| 3 | **SNS共有用サムネイル** | 横1200 × 縦630px | Instagram等でURLをシェアしたときに表示される画像 |

> 画像がまだない場合はデフォルト画像で仮公開し、あとから差し替えることも可能です。

## ご提供後にお客様ができること

管理画面（`/admin`）から以下の操作がブラウザ上で行えます。

| できること | 説明 |
|-----------|------|
| 商品の追加・編集・削除 | 商品名、価格、在庫数、画像を登録 |
| 商品画像のアップロード | 管理画面からドラッグ&ドロップ |
| 注文の確認・ステータス変更 | 入金確認、発送済みなどの管理 |
| CSV一括登録 | Excelなどで作った商品リストをまとめて登録 |
| お客様の声の管理 | レビュー画像の追加（トップページに表示） |
| お知らせバナーの管理 | セール情報などをサイト上部に表示 |

## 納品までの流れ

```
① このシートを記入して送付
　　　↓
② 内容を確認、不明点があればご連絡
　　　↓
③ サイト構築（1〜2営業日）
　　　↓
④ 確認用URLをお送り
　　　↓
⑤ 修正があれば対応
　　　↓
⑥ 本番ドメインに切り替えて公開
```

## よくあるご質問

**Q: 商品は何件まで登録できますか？**
A: 上限はありません。数千件でも問題なく動作します。

**Q: あとからカテゴリを追加できますか？**
A: はい。ご連絡いただければ追加いたします。

**Q: デザイン（色やレイアウト）の変更はできますか？**
A: 配色の変更は可能です。大幅なレイアウト変更は別途ご相談ください。

**Q: スマホでも見れますか？**
A: はい。PC・タブレット・スマートフォンすべてに対応しています。

**Q: ドメインを持っていないのですが？**
A: ドメインの取得・設定もサポートいたします。ご相談ください。

**Q: 月々の維持費はかかりますか？**
A: サーバー費用とドメイン費用が発生します。詳細は別途ご案内します。
